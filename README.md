# TNT Cart Cat Guard

Client-side Fabric mod for Minecraft 1.21.11 that warns the player about nearby TNT minecarts with a cat overlay, sound alert, and a toggleable keybind.

## Features

- shows a cat image when TNT minecarts are close
- toggle by default on `K`
- the key can be changed in Minecraft controls
- configurable radius, right-side position, size, colors, sound, and custom image path

## Config

The mod writes its config to:

`config/tntcartcatguard.json`

Set `imagePath` to an absolute path or a path relative to the game folder if you want to use your own PNG instead of the built-in cat image.
